# READ!!
<ins>Install instructions</ins>
1. run "git clone https://github.com/Coloured568/csinfo"
2. run "cd csinfo"
3. run "chmod +x build.sh"
4. run "./build.sh"
5. run "chmod +x ./csinfo"
6. run "./csinfo"
